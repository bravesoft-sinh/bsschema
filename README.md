# CentOSにて、構築ガイドライン
## 環境構築
> 前提条件：Java8以上がインストールされたこと

* この[リンク](http://www.graphviz.org/download/)でgraphvizをダウンロードして、インストールする
* 環境構築ツールをチェクアウトする
    ```
    git clone git@github.com:bravesoft-sinh/bsschema.git
    ```
* チェックアウトした`bschema`フォルダに移動して、`run.sh`に実行権限を追加する
    ```
    chmod +x run.sh
    ```
